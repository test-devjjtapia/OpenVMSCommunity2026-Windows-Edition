# Guías de Configuración

Este directorio contiene guías prácticas para configurar y solucionar problemas con OpenVMS Community 2026.

## Contenido

- **PUTTY-CONFIG.md** - Configuración de PuTTY para Windows
- **PUTTY-LINUX-CONFIG.md** - Configuración de PuTTY para Linux
- **TERMINAL-FIX.md** - Solución de problemas de terminal

## Cómo usar estas guías

1. Selecciona la guía correspondiente a tu sistema operativo
2. Sigue los pasos indicados
3. Si tienes problemas, consulta TERMINAL-FIX.md
